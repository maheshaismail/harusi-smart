# Harusi Smart Project Documentation

## Introduction
The Harusi Smart project aims to revolutionize the way we manage weddings and events through a comprehensive digital platform.

## Features
- **User-Friendly Interface**: Designed to ensure ease of use for all users.
- **Event Management**: Tools for scheduling, budgeting, and more.
- **Vendor Marketplace**: Access to vendors for all wedding needs.
- **Guest Management**: Easily manage RSVPs and guest lists.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/maheshaismail/harusi-smart.git
   ```
2. Navigate to the project directory:
   ```bash
   cd harusi-smart
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
- Access the application at `http://localhost:3000`
- Follow the on-screen instructions to navigate through the features.
